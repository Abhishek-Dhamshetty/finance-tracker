# 💰 FinanceTracker - Finance Dashboard UI

A modern, feature-rich finance dashboard built with React, TypeScript, and Tailwind CSS for managing personal finances.

**Built for:** Zorvyn FinTech Pvt. Ltd. - Frontend Developer Internship Assignment  
**Developer:** Abhishek Dhamshetty  
**Email:** abhishekdhamshetty@gmail.com

---

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Open **http://localhost:5173** in your browser

---

## ✨ Features

### Core Features
✅ **Dashboard Overview** - Financial summary with charts  
✅ **Transaction Management** - Add, edit, delete transactions  
✅ **Smart Filtering** - Search, filter, sort transactions  
✅ **Role-Based Access** - Admin vs Viewer modes  
✅ **Financial Insights** - AI-generated recommendations  
✅ **Dark Mode** - Full dark/light theme support  

### Bonus Features
📱 **Fully Responsive** - Mobile, tablet, desktop  
💾 **Data Persistence** - LocalStorage  
📊 **Rich Charts** - Line, pie, bar charts  
📥 **CSV Export** - Export filtered data  
⚡ **Fast Performance** - Optimized React app  

---

## 🛠️ Tech Stack

- **React 18** + TypeScript
- **Vite** - Build tool
- **Tailwind CSS v4** - Styling
- **Recharts** - Data visualization
- **React Router** - Navigation
- **Lucide React** - Icons
- **date-fns** - Date utilities

## 📁 Project Structure

```
src/
├── components/       # React components
│   ├── common/       # Reusable UI (Button, Card, etc.)
│   ├── dashboard/    # Dashboard widgets
│   ├── transactions/ # Transaction CRUD
│   ├── insights/     # Analytics
│   └── layout/       # Header, Sidebar, Layout
├── context/          # State management
├── hooks/            # Custom hooks
├── pages/            # Route pages

```
